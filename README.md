# Watchdog

Get WhatsApp alerts whenever someone approaches your unattended laptop.

## Prerequisites

Linux

    $ sudo apt-get install fswebcam

Mac OSX

    $ brew install imagesnap

## Installation

Requires NodeJS & NPM.

Once installed, clone the repository and install its dependencies running:

    $ npm install

## Development

Run:

    $ npm start

## Testing

For ESLint run:

    $ npm run lint

For tests run:

    $ npm run test
